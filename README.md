# AI Trade IDX - Phase 1 Scaffold

## Quick start

1. Copy env:
   - `cp .env.example .env`
2. Install deps:
   - `npm install`
3. Start infra + app:
   - `docker compose up -d`
   - `npm run dev`
4. Health check:
   - `curl http://localhost:3000/health`

## Available scripts
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run test`
- `npm run test:integration`
- `npm run smoke:phase1`
