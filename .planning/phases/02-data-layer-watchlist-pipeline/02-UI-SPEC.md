## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F3F0E8 | Page background and broad canvas surfaces |
| Secondary (30%) | #1F3A5F | Cards, table headers, tab active state, key status badges |
| Accent (10%) | #E06C2F | Primary CTA buttons, active run highlights, manual trigger emphasis |
| Destructive | #C62828 | Delete/destructive confirmations and hard error messaging |

Accent reserved for: `Save Watchlist Rules`, `Run Weekly Selection`, `Run Daily Data Pull`, `Retry Failed Symbol`, and success-path primary submit actions only.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Save Watchlist Rules |
| Empty state heading | No Watchlist Yet |
| Empty state body | Run the weekly selection to generate 5-10 LQ45 symbols, then review daily data readiness. |
| Error state | Data fetch failed for one or more symbols. Review provider status and retry the failed symbols. |
| Destructive confirmation | Remove symbol override: Remove this symbol override from weekly selection? This cannot be undone. |
